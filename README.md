# imersao-dev-study-tracker
O projeto "Dev Study Tracker" é uma ferramenta web minimalista desenvolvida usando HTML, CSS e JavaScript Vanilla (puro), que serve como um rastreador de metas de estudo no contexto da Imersão Dev.  O objetivo principal é ajudar o usuário a planejar e acompanhar seu progresso em tópicos de programação de forma visual e quantitativa.
